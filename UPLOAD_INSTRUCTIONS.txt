GitHub 上傳方式

1. 解壓縮 ZIP。
2. 打開解壓縮後的資料夾。
3. 將資料夾「裡面的全部檔案」上傳到 GitHub 專案最外層，不要再包一層資料夾。
4. 必須一起上傳隱藏項目：
   .github
   .env.example
   .gitignore
   .npmrc
5. GitHub 最外層應直接看到：server.cjs、package.json、package-lock.json、public、lib、test、scripts。
6. 提交後到 Render：Manual Deploy → Clear build cache & deploy。

Render 設定
Build Command：bash render-build.sh
Start Command：npm start
Health Check Path：/health

Gemini API 免費層
- Render Environment 填 GEMINI_API_KEY。
- GEMINI_MODEL 使用 gemini-3.6-flash。
- 不要把 API Key 寫進 GitHub、前端檔案或截圖。
